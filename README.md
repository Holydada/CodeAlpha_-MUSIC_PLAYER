# Music Player Project

## Features
- Upload music files
- Playlist creation and management
- Play, Pause, Skip
- Volume control
- Dark/Light mode toggle

## Tech Stack
- Frontend: React
- Backend: Node.js + Express

## Setup Instructions

### Backend
```bash
cd backend
npm init -y
npm install express multer cors
node server.js
```

### Frontend
```bash
cd frontend
npm create vite@latest .
npm install
npm run dev
```

## Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
